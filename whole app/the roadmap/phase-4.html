UI Design (الشكل)

الألوان

  • اختار 2–3 ألوان فقط.
  • اعمل Contrast واضح بين الخلفية والمحتوى.
  • ابعد عن الألوان العشوائية.

Typography

  • استخدم خط واحد أو خطين بحد أقصى.
  • اعمل hierarchy واضح (عنوان / نص / ملاحظات).

Spacing

  • Padding ثابت (مثال 16) و Margin منطقي.
  • المسافات هي الفرق بين مبتدئ ومحترف.

Layout

  • استخدم Grid و Alignment واضح.
  • التوازن البصري أهم من كثرة العناصر.

UX (تجربة المستخدم)

مبادئ أساسية

  • البساطة: قلل الخطوات.
  • السرعة: المستخدم عايز ينجز بسرعة.
  • الوضوح: كل زرار مفهوم.

مثال عملي

  • Login Screen فيها input واضح.
  • زرار واضح + رسالة خطأ مفهومة.
