What is a Database?

An organized collection of structured data, stored electronically for easy access and management.

مجموعة منظمة من البيانات المهيكلة، مخزنة إلكترونياً لسهولة الوصول والإدارة.

What is a DBMS?

Database Management System — the software that creates, manages, and controls access to databases.

نظام إدارة قواعد البيانات — البرنامج الذي ينشئ ويدير ويتحكم في الوصول لقواعد البيانات.

SELECT * FROM students WHERE gpa > 3.5;

SQL vs NoSQL

SQL: Structured Query Language for relational databases. NoSQL: Flexible schemas for documents, key-value, and graph data.

SQL: لغة الاستعلام المهيكلة. NoSQL: مخططات مرنة للمستندات والقيم والرسوم البيانية.
